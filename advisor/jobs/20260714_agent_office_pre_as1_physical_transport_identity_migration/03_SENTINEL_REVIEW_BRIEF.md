Use one independent `agent-office-reviewer` SOL Sentinel after the Worker result.
Load `/fable-sentinel`; if the CLI does not expose the slash command, directly
load and hash-verify the canonical skill file and only the relevant contract,
provenance, and classification references.

1. historical artifact immutability and identity lineage;
2. exact current target and live-observation correspondence;
3. fail-closed old/wrong/caller-selected destinations;
4. authority subject and evidence-stage consistency;
5. no transport, Slack, VibeNews, or external scope activation;
6. targeted tests and Worker report accuracy.

Verdict: `PASS`, `PASS_WITH_RISK`, `NEEDS_PATCH`, or `FAIL`.
