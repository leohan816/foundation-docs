It is not runtime source-of-truth.

Runtime behavior must be verified in the actual runtime repositories.

Each Advisor job should live under:

`jobs/<YYYYMMDD_project_stage_task>/`

- `00_INTAKE.md`
- `01_ADVISOR_BRIEF.md`
- `02_WORKER_BRIEF.md`
- `03_SENTINEL_REVIEW_BRIEF.md`
- `04_SERVICE_REVIEW_BRIEF.md`
- `05_FINAL_AUDIT.md`
- `index.md`
