## Strategist judgment on Round 3 wording

The Council converged substantively. Where exact wording differed, the Strategist
selects the smallest shared rule that prevents evidence promotion:

1. Rehearsal minimum includes separate Golden Order and captured-refund Golden Reversal,
   common contract/environment/configuration pins, end-to-end correlation, every-attempt
   reconciliation, server-authoritative values, relevant authorization, authentic and
   idempotent provider events, duplicate/out-of-order behavior, and immediate partial
   failure compensation for the selected loop.
2. A last-stock concurrency case is required before claiming the selected inventory
   path is safe, but it need not inflate the first Golden Order trace itself when the
   separately pinned immediate-integrity evidence closes it within the same rehearsal
   package.
3. Customer/operator-visible state and recovery traces accompany technical evidence,
   but they do not resolve the later U-018 human-readiness question.
4. `PASS` is always limited to a declared claim ceiling and never carries authority.
